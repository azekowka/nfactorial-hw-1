# nfactorial homework 1

## Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**

## Get started

1. install dependencies:
   ```bash
   npm i
   ```
2. run it on localhost:
   ```bash
   npm run dev
   ```
3. prod build check:
   ```bash
   npm run build
   ```
4. deployment:
   ```bash
   https://nfactorial-hw-1.vercel.app/
   ```
